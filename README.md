# Тест по фишингу

Готовый тестовый лист для Vercel: ученик выбирает класс, отвечает на 10 вопросов и отправляет ответы. Правильные ответы и баллы показываются только в админ-панели.

## Локальный запуск

```bash
npm run dev
```

Откройте `http://localhost:3000`.

## Админ

Кнопка `Админ` находится в правом верхнем углу. Локальный пароль по умолчанию:

```text
admin123
```

Для Vercel обязательно задайте переменную окружения:

```text
ADMIN_PASSWORD=ваш_сложный_пароль
```

## Vercel

1. Создайте проект на Vercel из этой папки.
2. В Environment Variables задайте `ADMIN_PASSWORD`.
3. Чтобы результаты сохранялись постоянно между запусками, подключите Vercel KV или Upstash Redis и задайте переменные:

```text
KV_REST_API_URL=...
KV_REST_API_TOKEN=...
```

Без KV локально результаты сохраняются во временный файл `/tmp/phishing-quiz-submissions.json`. На Vercel для настоящего сбора ответов нужен KV, иначе serverless-хранилище может очищаться.
